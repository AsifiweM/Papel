Are you using Papel yet?

What is Papel?

Papel is a light-weight core banking application that powers banking operations like account creation, customer deposits, and withdrawals. This app is meant to support a single bank, where users can signup and create bank accounts online, but must visit the branch to withdraw or deposit money.

How can I start using Papel?

Papel is easy to use. Just visit papel.io and enter your login credentials or click on the register button to create a new account.

What do I need to start using Papel?

To start using Papel, you just need a valid email. Browse to the register window and fill the form.

Does Papel offer entry credit?

Yes. For each new Papel Account that is created, Papel offers a credit amount of 10,000 RWF. This amount is found on the account.
